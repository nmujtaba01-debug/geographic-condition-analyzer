 
# Geographic Condition Analyzer

A beginner Python project that analyzes geographic data using loops, counters, AND conditions, and OR conditions.

## Features

- Count cities with population above 10 million
- Count cities with rainfall above 100 mm
- Count cities satisfying both conditions
- Count cities satisfying at least one condition

## Concepts Used

- Python Lists
- Dictionaries
- For Loops
- If Statements
- Counters
- AND Logic
- OR Logic
